pub fn snapshot_grid(grid: &[Vec<char>]) -> String
Expand description

Captures a grid snapshot as a formatted string.

This helper function converts a 2D character grid into a multi-line string representation suitable for snapshot testing or debugging. Each row becomes a line in the output string.

  • grid - Reference to a 2D grid (Vec<Vec> or similar)

use ratatui_testlib::shared_state::snapshot_grid;

let grid = vec![vec!['H', 'e', 'l', 'l', 'o'], vec!['W', 'o', 'r', 'l', 'd']];
